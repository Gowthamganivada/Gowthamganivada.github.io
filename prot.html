Smart Dustbin with Arduino

Developed a smart dustbin using Arduino and a servo motor that automatically opens the lid when an object is detected. The system enhances hygiene by enabling touchless operation.

Technologies: Arduino, Servo Motor, Ultrasonic Sensor, Embedded C

Role: Programmed the Arduino to control the servo motor and interfaced the ultrasonic sensor for object detection.

Impact: Participated in Lendi Institute’s College Project Expo, showcasing a practical IoT solution for waste management.

GitHub Repo

RFID Smart Gate

Built an automated smart gate system for cars using RFID technology, Arduino, and a servo motor. The gate opens automatically upon detecting an authorized RFID tag.

Technologies: Arduino, RFID Module, Servo Motor, Embedded C

Role: Designed the system, programmed the Arduino to interface with the RFID module, and controlled the servo motor for gate operation.

Impact: Demonstrated a secure and efficient access control solution for vehicle entry.
